Configuration Instructions

To add new presets, update your config.json configuration file: 

{
  "presets": [
    {
      "name": "custom_preset",
      "displayName": "Custom Preset",
      "headers": {
        "Authorization": "Bearer your-token",
        "X-Custom-Header": "your-value"
      },
      "headerMappings": {},
      "oidcConfig": {
        "enabled": true,
        "subject": "user@example.com",
        "clientId": "my-app-client",
        "redirectUri": "http://localhost:8080/callback",
        "scopes": ["openid", "profile", "email"],
        "tokenExpirationSeconds": 3600,
        "algorithm": "RS256",
        "keyId": "my-key-id",
        "email": "user@example.com",
        "name": "John Doe",
        "preferredUsername": "johndoe",
        "groups": ["users", "admin"],
        "customClaims": {
          "department": "engineering",
          "role": "developer"
        }
      }
    }
  ]
}
OIDC Configuration: When OIDC is enabled, the preset will provide OAuth 2.0/OIDC authorization endpoints and automatically generate JWT tokens with the configured claims.
